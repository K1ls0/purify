## Install

```
$ git clone https://github.com/banminkyoz/zero.git
$ cd zero 
$ cp zero.zsh-theme ~/.oh-my-zsh/themes/
```

## Usage

In your `.zshrc`, set `ZSH_THEME="zero"`. 

Restart terminal and enjoy.
